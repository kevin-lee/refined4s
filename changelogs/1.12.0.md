### Changes

#### [refined4s-core] Add better error messages for `Uri` and `Url` (#539)

* `Uri`
  ```scala 3
  Uri("%^<>[]`{}")
  ^^^^^^^^^^^^^^^^
  Invalid Uri value: Malformed escape pair at index 0: %^<>[]`{}
  ```

* `Url`
  ```scala 3
  Url("%^<>[]`{}")
  ^^^^^^^^^^^^^^^^
  Invalid Url value: no protocol: %^<>[]`{}
  ```
  ```scala 3
  Url("blah://www.google.com")
  ^^^^^^^^^^^^^^^^^^^^^^^^^^^^
  Invalid Url value: unknown protocol: blah
  ```
